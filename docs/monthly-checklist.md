## Quick Reference

### Target Metrics
- Organic traffic growth: +15% MoM
- Conversion rate: > 5%
- Average session duration: > 2 minutes
- Bounce rate: < 50%
- Core Web Vitals: All "Good"
- Average position: < 10 for target keywords
- CTR: > 3%

### Emergency Actions
If any metric drops > 30%:
1. Check Google Search Console for issues
2. Review Google Analytics for anomalies
3. Check site accessibility
4. Review recent changes
5. Investigate competitor activity
6. Escalate if needed
